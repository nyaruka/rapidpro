{% comment %} Shared persistence for the contact card column (contact read + ticket pages). Card order and collapsed state live in User.settings under "contact_cards" and are shared across pages. A page that shows only a subset of the cards merges its relative order into the saved full order so it never clobbers the position of cards it doesn't render. Requires a {{ card_settings|json_script:"card-settings" }} element and a temba-card-layout on the page; call initCardSettings(layout) on load. {% endcomment %}
